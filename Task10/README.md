# Muhammad Farzan — Portfolio Website

A responsive, animated personal portfolio built with HTML5, CSS3, Bootstrap 5 and vanilla JavaScript.

## 🗂️ File Structure
```
portfolio/
├── index.html          # All markup & sections
├── css/
│   └── style.css       # Custom design system (colors, type, layout, animations)
├── js/
│   └── script.js       # Interactivity (typing effect, scroll reveals, form, nav)
└── README.md
```

## ✨ Sections
- Hero / Home — animated role typing effect, code-editor visual
- About Me
- Skills — animated progress bars + tag cloud
- Projects — 4 featured projects with images, tags and links
- Experience / Education — dual timeline
- Contact — validated form with simulated send + social links

## 🛠️ Tech Used
- HTML5 / CSS3 (custom properties, Grid, Flexbox)
- Bootstrap 5 (grid & components, heavily re-themed)
- Vanilla JavaScript (IntersectionObserver for scroll reveals & counters, form validation, typing effect)
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- Bootstrap Icons

## 🚀 Deploy
This is a fully static site — no build step required.

**Netlify:** drag-and-drop the `portfolio/` folder onto Netlify, or connect the GitHub repo and set the publish directory to `/`.

**Vercel:** import the GitHub repo; framework preset "Other", no build command, output directory `/`.
